# 自由模型地编工具

这是 Godot 4.4 工程内的自由模型关卡编辑器原型，入口场景为 `res://scenes/level_editor.tscn`。

## 已支持

- 从 `res://assets/models/library/` 的分类资产库选择模型
- 资源库按 `C:\Users\zyh\Desktop\模型4.4` 原始目录结构镜像导入，保留 OBJ、MTL、贴图等相对引用关系
- 当前资源库排除 Blend、RAR，以及会引用缺失 `Untitled*.png` / `color palette.png` 的整包聚合模型
- 地编资产目录当前登记 332 个稳定 FBX 主模型，其中包含 5 个建筑资源，避免重复显示、Blend 依赖和材质路径错误
- 建筑资源的 Unity `.mat` 不直接用于 Godot，地编运行时会用同目录 `V1/V2/V3.png` 重建不透明双面材质
- 通过分类筛选模型，避免一次性显示全部资源
- 鼠标左键点到已有模型时选中模型
- 按住鼠标左键拖拽选中模型进行平移
- 选中物件后按住 `W` 上移，按住 `S` 下移，高度会随关卡保存/加载
- 鼠标左键点空地时自由放置当前模型
- 放置模型时自动将模型几何底部贴到地面
- 地形底板已扩大到 200 x 200，覆盖常规镜头范围
- 右键取消当前选中
- 旋转滑条控制当前画笔或选中模型旋转，滑条中点为 0 度
- 可以点击旋转数值手动输入角度，支持 `90` 或 `90°`
- 按住 `Q` 顺时针无级旋转，按住 `E` 逆时针无级旋转
- 缩放滑条控制当前画笔或选中模型缩放，范围 0.1 到 5 倍，1 倍为基准点
- 可以点击缩放数值手动输入倍数，支持 `1.5` 或 `1.5x`
- 按住 `Z` 无级缩小，按住 `X` 无级放大
- `Delete` 删除选中模型
- `Tab` 切换放置/选择模式
- UI 面板支持对选中模型进行旋转、缩放、归正、复制、粘贴、删除
- `Ctrl+C` 复制选中物件，`Ctrl+V` 粘贴物件，粘贴会保留位置高度、旋转和缩放并略微偏移
- 未选中物件时 `WASD` 平移视角，鼠标滚轮缩放
- 按住 `Alt` 并拖拽鼠标旋转镜头
- 按住 `Alt` 和鼠标中键拖拽平移场景
- 可切换 0.5 米位置吸附
- 保存/加载到 `user://levels/free_model_level.json`，保存内容包含物件位置、旋转、缩放、镜头缩放和镜头朝向
- 打开地编工具时会自动加载上次保存的 `free_model_level.json`
- 可导出 Godot 场景文件到 `res://scenes/exported_level.tscn`，导出内容包含当前物件、镜头、灯光、环境和地面，后续地编导出均使用 `.tscn`
- 一键生成自由摆放示例

## 首批模型

资产目录文件为 `res://data/asset_catalog.json`。

当前分类：

- 树木
- 岩石
- 植物
- 地形
- 建筑
- 木件
- 雪地树木
- 雪地岩石
- 雪地地形

## 关卡数据

每个摆放对象保存为 `asset_id`、`position`、`rotation_y`、`scale`。场景数据还会保存镜头 `distance`、`target` 和 `orientation`。后续要接引水逻辑时，可以在这个自由摆放层之外增加水路节点和连接数据。
